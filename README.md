## 📖 Sobre o Projeto
Este projeto foi desenvolvido durante o curso técnico de Informática, na disciplina de Programação Web. O objetivo era aplicar, na prática, conceitos de **HTML, CSS, JavaScript** e **orientação a objetos**.

A proposta consistia na criação de uma aplicação temática inspirada em um **RPG medieval**, seguindo requisitos específicos definidos em sala, como estrutura de telas e uso de lógica orientada a objetos.

---

## 🎮 Funcionalidades

O sistema é composto por três telas principais:

- **Tela Inicial**
  - Navegação para jogar, créditos ou sair
- **Tela de Jogo**
  - Seleção de classes
  - Estrutura baseada em RPG
- **Tela de Créditos**
  - Exibição dos desenvolvedores do projeto

---

## ⚔️ Classes do Jogo

O projeto implementa um sistema básico de classes, incluindo:

- 🛡️ Cavaleiro  
- 🗡️ Ladino  
- 🔮 Mago  
- (Outras classes podem ser adicionadas)

Cada classe pode possuir atributos e comportamentos próprios, permitindo expansão futura do sistema.

---

## 🧠 Conceitos Aplicados

- Estruturação de páginas com **HTML**
- Estilização com **CSS**
- Interatividade com **JavaScript**
- Uso de **Orientação a Objetos**
- Organização de telas e fluxo de navegação

---

## 📚 Objetivo Acadêmico

Este trabalho teve como foco consolidar o aprendizado prático em desenvolvimento web, integrando lógica de programação com construção de interfaces e simulação de sistemas simples.
